dvfy-htmx-table — server-side sort, search, paginate

Requires a server endpoint. Usage:

<dvfy-htmx-table src="/api/users" searchable>
  <table>
    <thead>
      <tr>
        <th data-sort="name">Name</th>
        <th data-sort="email">Email</th>
        <th>Actions</th>
      </tr>
    </thead>
    <tbody><!-- server-rendered rows --></tbody>
  </table>
</dvfy-htmx-table>
